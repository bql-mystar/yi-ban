yiban.py：
作用: 用于查看对应页数的投票中谁未投票，并返回一个txt文档
使用方法:
	start_url:打开易班网页班，进入投票页面，点开更多，复制对应的网页的url即可
	Cookie: 需要先登录易班，然后按F12，获取对应的Cookie传入
	student_list: 以列表的方式将全班成员的名称传入

发布易班投票.py：
作用：自动发布易班投票
使用方法:
	request_url_temp:打开易班网页班，进入投票页面，点开更多，复制对应的网页的url即可，寻找每一页对应的规律稍加修改即可
	Cookie: 需要先登录易班，然后按F12，获取对应的Cookie传入
	vote_title:需要发布投票的标题
	puid：可以从request_url_temp中轻易获取
	group_id:可以从request_url_temp中轻易获取
	subjectTxt_1:为对应的投票的第一个选项
	subjectTxt_2:为对应的投票的第二个选项
	subjectTxt_3:为对应的投票的第三个选项
